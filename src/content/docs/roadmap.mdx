## v0.1 - *The Fragrant Flower Blooms With Dignity* (in progress)

<details>
<summary>Core / Framework - 6 done</summary>

- [x] Full Havoc -> Mugen rebrand - theme, name, Python module
- [x] Themes - Mugen (sakura pink) + Havoc Classic (Dracula), switchable from navbar, persisted via QSettings
- [x] Agent ID prefixes - `dm-XXXXXXXX` (Demon), `tg-XXXXXXXX` (Tengu)
- [x] `make rebuild` - recompile teamserver + client without wiping cmake cache
- [x] DEADBEEF magic bytes randomization - unique value per server instance
- [x] djb2 HASH_KEY randomization - unique value per payload build

</details>

<details>
<summary>Tengu - Linux agent - 24 done, 3 remaining</summary>

- [x] HTTP/HTTPS C2, configurable sleep + jitter, configurable User-Agent
- [x] Base commands: `shell`, `sleep`, `exit`, `pwd`, `ls`, `cd`, `cat`, `download`, `upload`, `mkdir`, `rm`, `cp`, `chmod`
- [x] Process management: `ps`, `kill`
- [x] Identity: `whoami`, `whoami /all` (uid/euid, groups, capabilities, sudo rights, TTY, shell), `id`, `env`
- [x] Network recon: `netstat`, `arp`, `route`, `ifconfig` - parsed from `/proc`, no external binary
- [x] Port scanner - TCP connect scan, CIDR support, 64 parallel connections (`portscan`)
- [x] Credential harvester - SSH keys, cloud tokens (AWS/GCP/Azure), shell history, git, docker, kube, shadow (`harvest`)
- [x] Process memory scan - `/proc/<pid>/mem` without ptrace, credential pattern matching (`procdump`)
- [x] Keylogger - evdev raw input + X11 fallback, no external binary (`keylog`)
- [x] In-memory ELF execution - `memfd_create`, zero disk writes (`memfd`)
- [x] Persistence: `persist cron`, `persist systemd`, `persist bash`
- [x] Screenshot: X11 (scrot), Wayland (grim), ImageMagick (import)
- [x] SOCKS5 proxy over C2 channel
- [x] Reverse port forward - expose internal services without SOCKS5 (`rportfwd`)
- [x] ELF BOF loader - x86_64 in-process execution with full BeaconAPI (`inline-execute`)
- [x] `info` - local session metadata, no round-trip
- [x] `help` - tabular command list
- [x] DNS C2 transport - TXT record polling, base32-encoded frames in QNAME labels
- [x] DoH C2 transport - DNS-over-HTTPS POST `/dns-query`
- [x] TCP transport - outbound TCP, cross-platform pivoting (Windows Demon -> Linux Tengu)
- [x] ChaCha20 application-layer encryption - all frames encrypted post-INIT on all transports
- [x] Kill date + working hours - configured at payload build time
- [x] Explorer UI - process list + file explorer in Qt client
- [x] Desktop notifications on agent check-in (tray icon + toast)
- [ ] Sleep obfuscation - encrypt rwx segments between check-ins (eclipse/pendulum port)
- [ ] String obfuscation - XOR compile-time for hardcoded strings
- [ ] Privilege escalation recon - SUID/SGID, `sudo -l`, writable PATH, capabilities

</details>
